= Описание

Текстовое представление документа с сохранением форматирования.

= Принципы
 
- Однозначное представление сущности в виде текста.
- Наглядное представление.
- Простота парсинга/сериализации.
- Удобство редактирования в любой раскладке клавиатуры.
 
= Примеры синтаксиса
 
== Списки

- элемент списка
- ещё один элемент списка
  - вложенный элемент списка
    - многовложенный элемент списка
  - ещё один вложенный элемент списка

== Цитаты

== Стилизованный текст
 
- Это **акцентированный текст**.
- Это --удалённый текст--.
- Это ++добавленный текст++.
- Это  преформатированный текст  .

== Блоки кода

    Преформатированный
                 текст
  --Удалённая строка
  ++Добавленная строка
  **Выделенная строка
 
== Таблицы и сравнение с альтернативами
 
! **Язык**
  ! **Плюсы**
    ! **Минусы**
! MarkedText
  ! - Удобное редактирование таблиц.
  ! - Поддержка сложного форматирования внутри ячеек.
  ! - Простота реализации.
  ! - Легко запоминающийся консистентный синтаксис.
  ! - Удобство редактирования в русской раскладке.
  ! - Колонки не расползаются далеко вправо за горизонтальный скроллинг и не переносятся на новую строку так, что становится сложно понять, к какой колонке относится тот или иной текст.
    ! Не поддерживается пока что никакими сторонними инструментами
! MarkDown
  ! - Широкая поддержка различными инструментами
  ! - Наглядное представление таблиц
    ! - Сложности с редактированием таблиц
    ! - Сильно ограниченное содержимое ячеек
 
== Ссылки

- Кликабельный текст: \\**Содержимое** ссылки\http://example.org/\\
- Кликабельный урл: \\http://example.org/\\

- Кликабельная картинка с альтернативным текстом: ""текст\http://example.org/favicon.ico""
- ""http://example.org/favicon.ico""
- ""Интегрированное видео\https://youtube.com/video=1234""
- ""Интегрированный сайт\https://hyoo.ru""

- \\""**Альтернативное** содержимое\http://example.org/favicon.ico""\http://example.org/\\
